# Multiplicar STR
```py
    hola = "Hola"
    print(hola * 3)
```
### Salida
```py
    "HolaHolaHola"
```

# Multiplicar Vector
```py
    hola = ["Hola"]
    print(hola * 3)
```
### Salida
```py
    ["Hola", "Hola", "Hola"]
```

# Inverir Vector
```py
    hola = [1,2,3]
    print(hola[::-1])
```
### Salida
```py
    [3, 2, 1]
```
